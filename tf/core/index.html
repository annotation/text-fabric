Module tf.core

Core API of TF

This API deals with the basic TF data model: a graph of nodes and edges, annotated by features.

The core API consists of

  • N: see Nodes (walk through nodes)
  • F: see NodeFeature (retrieve feature values for nodes)
  • E: see EdgeFeature (retrieve feature values for edges)
  • L: see Locality (move between levels)
  • T: see Text (get the text)
  • S: see tf.search.search (search by templates)

plus some additional methods.
