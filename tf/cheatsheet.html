Layered search

(these are commandline instructions)

text-fabric-make {dataset} {client} ship

generate a static site with a search interface in client-side javascript and publish it to GitHub pages. If {client} is left out, generate all clients that are defined for this dataset. Clients are defined in the app-{dataset} repo, under layeredsearch.

text-fabric-make {dataset} serve

serve the search interfaces defined for {dataset} locally.
